Cheat Sheet

HTML Table Reference

Table Tags

Tag Name Description
<table> Table The wrapper for HTML tables.
<thead> Table Head Defines head section inside of table.
<tbody> Table Body Defines body section inside of table
<tfoot> Table Foot Defines footer section inside of table.
<th> Table Header Defines a set of rows defining the head of the columns of the table.
<tr> Table Row Adds row to a Table
<td> Table Data Adds data to a Table.

Table Attributes

Attribute Name Description
colspan Column Span Defines how many rows an element should span.
rowspawn Row Spawn Defines how many columns an element should span.
